Methodology. The 3-bucket strategy withdraws cash first (Bucket 1), then bonds (Bucket 2), then stocks (Bucket 3). After each year's growth, Bucket 1 is refilled from Bucket 2 toward the target years-of-expenses, and Bucket 2 is refilled from Bucket 3 similarly. This reduces sequence-of-returns risk by letting equities recover during downturns. The FIA bucket is modeled as a separate sleeve — when an income rider is enabled, the income base grows at the roll-up rate until activation age, then pays the payout rate as a guaranteed lifetime income stream (the underlying account value still draws down if withdrawals exceed it). Pre-65 healthcare assumes ACA / COBRA pricing; post-65 uses your Medicare estimate. Returns are deterministic — for sequence risk, run the FIA-Heavy preset to see how guaranteed-income flooring changes survival.
Educational tool — not financial advice. Actual outcomes depend on market sequence, tax structure (Roth vs traditional), state tax (TX vs MA matters), Medicare IRMAA brackets if income is high, and unmodeled events (LTC, market crashes, divorce, family support). Always validate against Monte Carlo simulators (e.g., FIRECalc, cFIREsim) and a fee-only fiduciary CFP before acting. Built to run fully offline — no telemetry, no external calls beyond the Chart.js CDN.
